What this tool is for:

This tool was made to allow people to easily implement a lightweight alternative to the embeddable code that YouTube provides.

The idea for this tool stemmed from an obsession to optimise websites for lightning fast page speeds; You can read about it in the blog post here.

This is a responsive tool and will work at any screen size. It will significantly reduce the initial load time of your web page and make for a better experience on mobile with a 3G or 4G connection.
